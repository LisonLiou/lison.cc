记一次失败经历 Arduino UNO 使用 L298N 驱动直流电机

Lison
2023-05-14 (Updated: 2023-11-16)
arduino
arduino

打算是要给多年前的遥控玩具车完整的一生，尸体只保留了车架子跟两个马达，后驱的车轴部分跟轮胎都找不到了，先上一段失败的成品演示。

需求

  1. 使用已经吃灰的一个电视盒子遥控器发射红外信号（NEC协议）
  2. 接收到红外指令后可以完成前后左右四个方向动作

配件

  1. Arduino Uno
  2. L298N驱动板
  3. 两个直流小马达
  4. 红外接收头1838一只（从吃灰的DVD上拆的。。）
  5. 某支持NEC协议的红外遥控器一个
  6. 杜邦线
